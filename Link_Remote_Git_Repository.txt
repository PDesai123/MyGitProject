
your remote repository is your GitHub Repository
Your local repository is on your local windows machine.

-----------------------------------
--        GitHub Actions         --
-----------------------------------

[1] First logon to GitHub and click on your repository "MyGitProject"

[2] Click "Code" => and "SSH" in popup menu => click Clip-Board icon next to URL in textBox to copy SSH URL. 
We will use this as Remote URL while linking remote Repository to your local repository


-----------------------------------
-- On your Local Windows machine --
-----------------------------------

[1] Go to command prompt on your windown machine and go to your Local Git Repository folder.

[2] To add a new remote, use the git remote add command on the terminal, in the directory your repository is stored at

get remote add <Unique_Remote_Name>   <Remote_URL>

example
	git remote add MyGitProject git@github.com:PDesai123/MyGitProject.git

[3] verify linked remote repository using following command

	git remote -v

output
------
	D:\Cloud\VersionControl\GIT\Git-Repository>git remote -v
	MyGitProject git@github.com:PDesai123/MyGitProject.git (fetch)
	MyGitProject git@github.com:PDesai123/MyGitProject.git (push)
