
How To Create Create Local Git Repository
=========================================

[1] Create a folder for local repository on your local widnows machine.

[2] Open command prompt (make sure you are in folder you created for local git repository)

[3] Initialize git repository using command "git init"

D:\Cloud\VersionControl\GIT\Git-Repository>git init
Initialized empty Git repository in D:/Cloud/VersionControl/GIT/Git-Repository/.git/

[4] this will create a ".git" subfolder. 


you have intialized your local git repository. All changes are in this folder are tracked.
